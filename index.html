Acerca del Proyecto

Este dashboard de visualización de datos es un proyecto de muestra diseñado para demostrar mis habilidades en:

  • Desarrollo front-end con HTML, CSS y JavaScript
  • Implementación de Bootstrap para diseño responsivo
  • Creación de interfaces de usuario interactivas
  • Visualización de datos con Chart.js
  • Implementación de modo claro/oscuro
  • Navegación fluida con navbar sticky

Los datos mostrados son ficticios y se utilizan únicamente con fines demostrativos.
